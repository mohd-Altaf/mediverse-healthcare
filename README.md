# Mediverse Healthcare

Welcome to the official repository for **Mediverse Healthcare**.

🌐 **Live Website:** https://mediversehealthcare.com

## About

Mediverse Healthcare is a fully optimized healthcare website built to provide a professional digital presence for medical services, patient information, and healthcare solutions.

The project is designed with performance, responsiveness, SEO readiness, and user experience in mind.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

* `index.html` — Homepage
* `about.html` — About Us page
* `services.html` — Services page
* `gallery.html` — Media / Gallery page
* `contact.html` — Contact page
* `style.css` — Styling and layout
* `script.js` — Interactive functionality
* `images/` — Website media assets
* `CNAME` — Custom domain configuration
* `sitemap.xml` — SEO sitemap
